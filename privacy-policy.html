Meeting Browser Alert

Politica de Privacidade

Ultima atualizacao: 10 de maio de 2026

Visao Geral

Meeting Browser Alert ajuda o usuario a perceber reunioes proximas diretamente no navegador. A extensao le eventos do calendario conectado apenas para calcular alertas visuais e permitir acesso rapido ao link da reuniao.

Dados Acessados

Quando o usuario conecta o Google Calendar, a extensao pode ler:

  • titulo do evento;
  • horario de inicio e termino;
  • link de reuniao, quando disponivel;
  • informacoes basicas necessarias para identificar a plataforma da reuniao.

A extensao nao solicita permissao para criar, editar ou excluir eventos.

Como Os Dados Sao Usados

Os dados sao usados para:

  • mostrar o proximo compromisso no popup;
  • exibir um alerta visual em paginas abertas;
  • abrir o link da reuniao quando o usuario clica em Entrar;
  • lembrar o usuario quando uma reuniao ja iniciou dentro da janela configurada.

Armazenamento

As configuracoes, cache temporario de eventos, alertas dispensados e preferencias ficam no navegador do usuario usando chrome.storage.local.

Compartilhamento De Dados

Meeting Browser Alert nao vende dados, nao compartilha dados com terceiros, nao usa analytics e nao envia eventos de calendario para servidores proprios nesta versao.

Permissoes Do Navegador

A extensao usa:

  • identity para conectar ao Google Calendar com OAuth;
  • storage para salvar preferencias locais;
  • alarms para verificar periodicamente reunioes proximas;
  • tabs para detectar se o link de uma reuniao ja esta aberto;
  • offscreen para tocar alertas sonoros curtos;
  • acesso a https://www.googleapis.com/* para ler eventos do Google Calendar;
  • content script em paginas http e https para exibir o alerta visual.

Controle Do Usuario

O usuario pode:

  • pausar os alertas;
  • desativar som;
  • desativar overlay e efeitos visuais;
  • trocar o idioma;
  • dispensar alertas;
  • desconectar o Google Calendar.

Contato

Para duvidas de privacidade ou suporte, use o canal de contato informado na Chrome Web Store.
