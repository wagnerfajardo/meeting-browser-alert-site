Product Use

Meeting Browser Alert is a browser extension that helps users notice upcoming meetings through local alerts in the browser.

Google Account And Permissions

Some features depend on voluntarily connecting Google Calendar. By connecting a calendar, the user authorizes only the access needed to read events used by the extension's alerts.

User Responsibility

  • keep access authorized only when they want to use the calendar integration;
  • review alert settings according to their routine;
  • use the extension according to applicable laws and policies.
